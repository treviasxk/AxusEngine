## 2D and 3D cross-platform game engine

**Axus Engine** is a fork of [Godot Engine](https://godotengine.org) focused on **delivering advanced features faster**, especially modern rendering technologies such as ray tracing and path tracing (through the NVIDIA RTX fork).

It keeps the unified interface philosophy of Godot, but includes **small modifications to the editor interface** to make the developer's workflow even smoother and more intuitive, while accelerating the integration of cutting-edge features.

Axus provides a comprehensive set of common tools, so you can focus on making games instead of reinventing the wheel. Games can be exported with one click to a number of platforms, including the major desktop platforms (Linux, macOS, Windows), mobile platforms (Android, iOS), Web-based platforms, and [consoles](https://godotengine.org/consoles).

## Free, open source and community-driven

Axus Engine is completely free and open source under the very permissive **[MIT license](https://godotengine.org/license)** — the same license as Godot.  
No strings attached, no royalties, nothing. Your games belong to you, down to the last line of engine code.

This project is a fork of Godot Engine and fully respects its original license. All original Godot code retains its copyrights and terms.
